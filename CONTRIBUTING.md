# Contributing

## Branching Model
- `main` is protected — never commit directly
- Create a feature branch for each story: `feature/story-name`
- Open a Pull Request to merge into `main`

## Commit Convention
Use conventional commits:
- `feat:` new feature
- `fix:` bug fix
- `chore:` setup, config, tooling
- `docs:` documentation only

## Example
feat: add llm_client module with Claude and OpenAI support
